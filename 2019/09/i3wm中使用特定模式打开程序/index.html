i3wm中使用特定模式打开程序

九月 9, 2019 发布在 随便写写

有时候我们不想某些程序按照平铺方式打开，比如说telegram、网易云音乐，我们希望打开这些应用的时候直接使用悬浮模式(floating)，这时候就需要使用for_window命令。

在官方的userguide中是这么描述说的：

With the for_window directive, you can let i3 execute any command when it encounters a specific window. This can be used to set windows to floating or to change their border style, for example.

Syntax:

for_window <criteria> <command>

Examples:

# enable floating mode for all XTerm windows
for_window [class="XTerm"] floating enable

# Make all urxvts use a 1-pixel border:
for_window [class="urxvt"] border pixel 1

# A less useful, but rather funny example:
# makes the window floating as soon as I change
# directory to ~/work
for_window [title="x200: ~/work"] floating enable

更多细节可以查看 for_window

比如说我们想让telegram打开的时候直接是悬浮模式(floating),我们可以在配置文件(~/.config/i3/config)中加上这么一句：

for_window [class="TelegramDesktop"instance="telegram-desktop"] floating enable

class表示窗口类，instance表示窗口实例，如果不清楚的话，可以先打开程序，然后在终端输入

$ xprop

然后选中那个程序，终端就会输出这个窗口的信息，以telegram为例：
