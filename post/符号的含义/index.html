'%'符号的含义

’%‘符号有两个含义–取模运算（“Modulo Operation”）和取余运算（“Complementation ”）。两个概念有重叠的部分但又不完全一致。主要的区别在于对负整数进行除法运算时操作不同

含义

不同语言，’%‘的含义不同

语言 含义
C/C++ 取余
C# 取余
Java 取余
Python 取模
Golang 取余
Rust 取余

计算方法

对于整数a, b(b!=0),计算r = a%b有两个步骤;

1.c = a/b

2.r = a - c*b

求模运算和求余运算在第一步不同: 取余运算在取c的值时，向0 方向舍入(fix()函数)；而取模运算在计算c的值时，向负无穷方向舍入(floor()函数).

a b c(取余) c(取模) rem mod
4 3 1 1 1 1
3 4 0 0 3 3
4 -3 -1 -2 1 -2
-4 3 -1 -2 -1 2
3 -4 0 -1 3 -1
-3 4 0 -1 -3 1
-4 -3 1 1 -1 -1
-3 -4 0 0 -3 -3

应用

判别奇偶数

已知一个整数n对2取模，如果余数为0，则表示n为偶数，否则n为奇数。

bool IsEven(int n)
{
    return (0 == n%2);
}

辗转相除法（欧几里德算法）

辗转相除法是求最大公约数最常见的方法，其计算原理依赖于定理：gcd(a,b) = gcd(b,a mod b)

int gcd(int c, int y)
{
    return y ? gcd(y, x%y) : x;
}
