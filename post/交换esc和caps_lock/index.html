交换Esc和Caps Lock

不管是不是vim党，Esc的使用频率都远远高于Caps Lock键，但是Esc键太远，反而Caps Lock键占着非常棒的位置，这谁能忍？所以我们可以交换这两个键。

环境

archlinux + xorg + dde

各种方案

xmodmap

  1. 首先安装xmodmap,在arch上包名是xorg-modmap

  2. 在你的HOME目录下新建一个文件，例如.Xmodmap， 写入以下内容:

    remove Lock = Caps_Lock
keysym Escape = Caps_Lock
keysym Caps_Lock = Escape
add Lock = Caps_Lock

  3. 然后在你的.xprofile加上这一行

    xmodmap ~/.Xmodmap

然后注销或者source一下就好了，不过这种方案在锁屏之后就会失效。

利用xkb

我是修改了/usr/share/X11/xkb/keycodes/evdev文件，将Esc和Caps Lock键的keycode互换了，不过和上面方案一样，锁屏或注销之后就不行了

更改xorg的键盘配置

在/etc/X11/xorg.conf.d/00-keyboard.conf增加Option "XkbOptions" "caps:swapescape"

这个方案目前未发现任何问题
