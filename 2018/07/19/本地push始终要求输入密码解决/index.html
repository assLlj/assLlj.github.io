前言

最近在本地提交代码的时候总是要求输入密码，但是我已经丢了公钥在Github了，检查后发现是因为我clone的方式是https

clone的两种方式

我们在Github上复制项目的URL的时候，可以看到它会有显示
Clone with HTTPS
或者
Clone with SSH

这两者的区别如下:

HTTPS SSH
URL https://github.com/username/repositoryname.git git@github.com:username/repository.git
Clone 无限制 无限制
Pull 无限制 无限制
Push 必须输入用户名和密码 必须在远程仓库提交了本地的公钥，无需输入密码（生成密钥时使用了密码则需要）
